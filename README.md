# RFID Registration Project

## Overview
The RFID Registration Project is a web application designed to facilitate the registration of individuals using RFID technology. The application provides a user-friendly interface for capturing personal information, including RFID ID, role, and a profile photo.

## Project Structure
```
rfid-registration-project
├── public
│   ├── rfid_registration.html   # HTML structure for the RFID registration form
│   └── styles.css               # CSS styles for the registration form
├── src
│   └── js
│       └── rfid_registration.js  # JavaScript logic for handling form functionality
├── package.json                  # npm configuration file
├── .gitignore                    # Git ignore file
└── README.md                     # Project documentation
```

## Features
- **RFID Registration Form**: Users can input their RFID ID, select their role, and provide personal information.
- **Profile Photo Capture**: Users can capture a profile photo using their device's camera.
- **Photo Cropping**: Users can crop their captured photo before submission.
- **Preview Functionality**: Users can preview the ID created from the input data before final submission.

## Getting Started
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd rfid-registration-project
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Open `public/rfid_registration.html` in a web browser to access the registration form.

## Usage
- Fill in the RFID ID and select the appropriate role.
- Enter personal information in the provided fields.
- Click on the "Open Camera" button to capture a profile photo.
- After capturing, crop the photo as needed.
- Use the "Preview" button to see a preview of the ID created from the input data.
- Finally, click "Submit" to register.
